# streaming-service

Service de streaming vidéo en direct

## Installation

```bash
npm install
```

## Démarrage

```bash
# Mode développement
npm run dev

# Mode production
npm start
```

## Tests

```bash
npm test
```

## API

- **Port**: 3005
- **Health Check**: http://localhost:3005/health
- **Documentation**: http://localhost:3005/api/docs

## Variables d'environnement

Copiez `.env.example` vers `.env` et configurez les variables.
